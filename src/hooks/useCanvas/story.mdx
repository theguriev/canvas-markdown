# useCanvas

Utility hook to draw something into canvas

## Usage

```js
import { useCanvas } from 'canvas-markdown'

useCanvas({
    dom,
    before: () => { /* Will be launched before starting */ },
    draw: ({ context, dom, counter }) => { /* Will be launched on each frame */ },
    after: () => { /* Will be launched after stop */ }
})
```
