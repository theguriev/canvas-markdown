# usePan

Used to move drawings around the canvas

## Usage

```js
const {
    isMouseDown,
    isSpacePressing,
    offsetX,
    offsetY,
    dom
} = usePan({
    dom, // canvas DOM element
    offsetX: { current: 0 }, // start offsetX
    offsetY: { current: 0 }, // start offsetY
})
```
