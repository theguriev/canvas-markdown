# usePosition

Used to calculate mouse position on the canvas.

## Usage

```js
const {
    x,
    y,
    hoverX,
    hoverY
} = usePosition({
    dom, // canvas DOM element
    offsetX: { current: 0 }, // start offsetX
    offsetY: { current: 0 }, // start offsetY
})
```
