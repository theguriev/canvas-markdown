# useZoom

Utility hook to zoom the canvas

## Usage

```js
useZoom({
    dom,
    startScale, // start scale number
    startX = 0, // start X
    startY = 0, // start Y
    maxZoom = 50, // Max zoom level
    zoomStep = 0.2, // Zooming step
    minZoom = 0.25 // Min zoom level
})
```
